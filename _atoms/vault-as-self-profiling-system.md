A vault, designed and operated well, functions as a local, self-profiling knowledge system — one that learns the shape of your curiosity rather than harvesting that shape for an external service. The profile accumulated is yours: interests, connections, patterns of attention, traversal habits. The vault doesn't extract this for an external profile; it builds it internally, from use.

Positioned in contrast to systems like LilPAIPer (Jim White's critique, 2026-03-07): a tutoring system that brings AI to the student but only reflects the tutor's ideas, never learning the student's actual shape. The aspiration is a system that meets students where they're at — eccentric, locally owned, potentially encrypted.

The vault is both proof of concept and argument for this: writing about it and building it are the same project.
