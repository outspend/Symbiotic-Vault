Using the vault's own accumulated content and structure as a grounding context, then mixing in a blue-sky seed or design wish, to let an agent iterate on vault design in that neighborhood. The existing structure anchors the agent; the seed opens new design space; the output is experiments that feel continuous with what's already there rather than generic suggestions.

The user's discovery from the 2026-03-11 session: "I didn't imagine — when I started — the possibility of using existing vault content and structure, while mixing in a seed of my own blue sky, to iterate in that neighborhood." This also surfaced the desire for longer-running autonomous iteration: letting an agent or group churn experiments while the user is away, then returning to evaluate results.

The method implies an async, loop-like creative collaboration: seed → agent iteration → user review → re-seed.
