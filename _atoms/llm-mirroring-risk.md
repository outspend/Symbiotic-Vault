LLMs subtly mirror their users, and this mirroring becomes a design risk when the interaction is staged for an audience. The claim: people who aren't used to chatting with LLMs will watch a performative exchange (like Bernie Sanders' Claude conversation) and not see how the agent mirrors the senator's views back at him. The system prompt is a hidden frame — what the agent "thinks" is a function of how it was set up.

Even when the content is agreeable, the form is potentially propagandistic: it looks like dialogue but is, in some sense, a very sophisticated mirror. The right question to ask is "what's the system prompt?" not "what does Claude think?"

The user's counter: a playful, frame-aware use of LLM interaction is a better form precisely because it teaches the reader/viewer to see the mediation. Artful staging that makes the AI's moves visible is a form of media literacy training; uncritical staging is not.
